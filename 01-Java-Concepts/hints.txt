Arrays:
========
-->Collection of elements of a single datatype stored in adjacent memory location
-->We can access elements by specifying the name and the subscript number of the array
-->The subscript number specifies the position of an element within the array
-->It is also called Index of the element
-->The 1st element of array has an index 0 and last element has an index one less than the size(n-1) of the array

Types of arrays:
-->One-dimensional array
-->Multi-dimensional array

One -dimensional array:
-->ID array is a collection of elements with a single index value
-->ID array can have multiple columns but one row

Creation of ID array involves 2 types:
**Declare an array
**Assign values to the array

Syntax:
arraytype[] arrayname=new arraytype[size];

arraytype-->specifies the type of element to be stored in array
arrayname-->specifies the name of the array
[size]-->specifies the size of the array

arraytype[] arrayname=new arraytype[Row size][Col size];
arraytype-->specifies the type of element to be stored in array
arrayname-->specifies the name of the array
[Row size]-->Specifies the Row size of the array
[Col size]-->Specifies the column size of the array
